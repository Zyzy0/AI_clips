## Uruchomienie
* Sklonowanie repozytorium lub pobranie plików
```
git clone https://github.com/Zyzy0/AI_clips.git
cd AI_clips
```
* Przygotowanie środowiska
```
# Windows
python -m venv .venv
.venv\Scripts\activate

# Linux / MacOs
python3 -m venv .venv
source .venv/bin/activate
```
* Pobranie wymaganych bibliotek w (.venv)
```
pip install -r requirements.txt
```
* Uruchomienie programu w (.venv)
```
python main.py
```
